Delete Characters to Make Fancy String

Difficulty: Easy

A fancy string is a string where no three consecutive characters are equal. Given a string s, delete the minimum number of characters to make s fancy. Return the final string after the deletion.

Example 1:

Input: s = "leeetcode"
Output: "leetcode"
Explanation: Remove one 'e' to avoid three consecutive 'e'.

Example 2:

Input: s = "aaabaaaa"
Output: "aabaa"
